SINTAXIS CSS

Explicación del Modelo de Caja

El modelo de caja está conformado por:

  • Contenido: Es el área donde se muestra el texto, imágenes u otros elementos.
  • Padding: Es el espacio entre el contenido y el borde. (Ejemplo: 80px)
  • Borde: Es la línea que rodea el contenido y el padding. (Ejemplo: 20px de grosor y con color definido)
  • Margen: Es el espacio exterior que separa la caja de otros elementos. (Ejemplo: 50px)
